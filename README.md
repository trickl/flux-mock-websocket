Utility library for testing reactive websockets

Installation
============

To install from Maven Central:

```xml
<dependency>
	<groupId>com.github.trickl</groupId>
	<artifactId>flux-mock-websocket</artifactId>
	<version>0.0.2</version>
</dependency>
```
